# 專精系統

專精積分僅能透過贏得比賽或成為「MVP」而獲得。每個英雄都有自己的專精積分。如果玩家是 3v3 或 5v5 比賽中的「MVP」，則會額外獲得 30% 的專精積分。當英雄在專精系統中升級時，玩家可獲得金幣、能量點數、英雄幾分、捂臉表情、第二個英雄玩家頭像或稱號。
